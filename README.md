# filterjs
 
## how to use

download the js file in our dist folder

```html
<script src="path/to/file/filterjs.min.js"></script>
```

## Supported tags
- [x] input

## Use as html attributes
- filter="{dataType}"

## Available data type

- [x] integer
- [x] decimal
- [x] alpha
- [x] email
- [x] boolean
- [x] alphadash

## Examples

``` html
<input filter="alpha">
```

## Use multiple data type

<input filter="integer|alpha">
